This is a crawler to gather all bonus information of the most famous Dutch supermarket, Abert Heijn.

#Prerequisites
To run this crawler, you need to install the following software/pakages:

1. Python 3.5.x

2. Selenium Package for Python

3. lxml for Python  

4. Chrome

5. Chrome WebDriver
